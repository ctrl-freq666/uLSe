# µ-LSe (micro Light Sequencer)

µ-LSe is a pocket-sized, Arduino-based sound device that turns light into sound.

It’s a stripped-down, workshop-friendly version of the LSe (Light Sequencer) — designed as an accessible entry point into DIY electronics, experimental instruments, and creative coding.

## What it does

* Generates light patterns using RGB LEDs
* Reads light with photo sensors
* Translates light into sound using simple algorithms (bytebeat / PWM synthesis)
* Lets you interact via knobs, sensors, and buttons

## Features

* 2x RGB LEDs
* 2x light sensors (LDRs)
* 2x buttons (mode switching)
* Potentiometer control
* Piezo / mini speaker + jack output
* Multiple sound and light modes

## Why?

The original LSe is powerful but time-consuming and expensive to build.
µ-LSe keeps the core idea — light controlling sound — but reduces it to something you can build in a couple of hours.

It’s meant to be:

* hacked
* modified
* broken
* rebuilt

## Workshops

µ-LSe was designed with workshops in mind, as a hands-on introduction to:

* Arduino
* DIY electronics
* experimental instrument design
